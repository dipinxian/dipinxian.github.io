####Top1

  • 此命令打印出操作系统名字,版本,xnu版本,cpu型号等.
  • xnu版本可在https://opensource.apple.com中查找.

####Top2

  • 以上命令打印出Mac 硬件信息.
  • system_profiler可以命令打印Mac OSX系统硬件和软件的配置
  • listDataTypes 在man system_profiler命令中查找
  • SPHardwareDataType 参数在 system_profiler -listDataTypes 命令中查找
    ####Top3
  • 以上命令打印出 cup的商标
  • sysctl 设置或得到内核的状态或者打印硬件信息.
  • machdep.cpu.brand_string 代表cup商标,在man sysctl 命令中可以找到.

####Top4

  • launchctl命令是用来和内核进程launchd交互的.可以操纵launchd的启动,停止.
  • Mac OS X系统启动过程:BIOS自检->加载内核->加载launchd->启动系统级别守护进程和代理程序->创建终端->登陆->启动用户级别守护进程和代理程序
  • services 就是各种守护程序和代理程序.

####Top5

  • 此命令允许任何来源的应用程序安装到电脑中.系统默认是不允许的,有时安装应用程序需要使用到
  • spctl命令用于设置子评估系统的策略安全/是否可用

####Top6

  • 此命令用于配置网络接口参数.查看IP地址和Mac地址.
  • 直接使用ifcofig命令,在en0的位置可以显示Mac的网卡,IP,Mac地址等.
  • Mac地址:Mac不是指Mac电脑,而是Media Access Control地址,指的是网卡地址,用6字节十六进制表示.一台设备如果有一个或多个网卡,则每个网卡都有一个唯一的Mac地址.
  • IP地址和Mac地址是由ARP协议一一映射的
    ####Top7
  • ping命令一般用于测试网络/检测主机是否可以访问.它是使用ICMP协议,发出请求,如果主机能连通,会收到响应.如果不通,会提示cannot resolve host / time out.
  • c是count,代表打印到终端的次数
  • s为size,代表每次传输多少字节,默认是56个字节

####Top8

  • top命令用于实时显示进程状态
  • ps aux | grep qq打印的是qq进程的行信息,包括USERPID,%CPU ,%MEM,VSZ,RSS,TT,STAT,STARTED,TIME等KEYWORDS,这些KEYWORDS可以通过ps aux 命令查看到,并且man ps命令均可查看各字段的意思.

####Top9

  • dscl命令用于用户和用户组的管理,均可以添加,删除,修改用户和用户组.

####Top10

  • 以上两个命令可以实现局域网内的通讯.
  • 两台电脑的端口必须一致
  • 当然还可以实现局域网文件传输,类似于AirDrop功能.
